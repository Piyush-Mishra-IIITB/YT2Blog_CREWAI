# YouTube Video Blog Generator

This project automates the process of creating blog posts from YouTube videos. Using AI agents, it can research video content and generate a detailed blog post based on a given topic.

## Features

- **Video Research:** Extracts detailed information from YouTube videos related to a given topic.
- **Blog Writing:** Summarizes video content and generates a polished blog post.
- **Automation:** Uses CrewAI framework with custom agents and tools for end-to-end content creation.
- **Output:** give the blog
